# About Project
Парсер пользователей из VK

## Требования

* PHP >= 8.0
* Composer >= 2
* make >= 4

## Install and start project
* `host write 127.0.0.1 user-parserl.ru`
* `make setup`


## Tests and lint

* `make lint`
* `make lint-fix`
* `make test`
* `test-coverage`
